Pertemuan 02 - Titik dan Palet Warna

Membaca dan Menulis Pixel

Piksel (picture element) adalah unit terkecil dalam gambar digital. Setiap piksel memiliki informasi warna yang ditentukan oleh kombinasi warna dasar (RGB).

Konsep Dasar Pixel

Resolusi gambar diukur dalam PPI (Pixels Per Inch). Semakin tinggi PPI, semakin tajam dan detail gambar yang dihasilkan.

PPI = √(Jumlah Pixel Horizontal² + Jumlah Pixel Vertikal²) / Diagonal Layar (inci)

Contoh: Monitor 24 inci dengan resolusi 1920×1080 memiliki PPI ≈ 92

Model Warna RGB dan HSV

RGB (Red, Green, Blue)

Model warna aditif untuk perangkat elektronik. Setiap warna direpresentasikan dengan kombinasi nilai merah, hijau, dan biru (0-255).

HSV (Hue, Saturation, Value)

Model warna yang lebih mendekati persepsi manusia dengan komponen Hue (warna), Saturation (kekuatan warna), dan Value (kecerahan).

Konversi RGB ke HSV

Input RGB

Output HSV

Hue (H): 0°

Saturation (S): 0%

Value (V): 0%

Rumus Konversi

H = Sudut warna (0°-360°)

S = Saturasi (0-100%)

V = Kecerahan (0-100%)
